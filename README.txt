// This document explains how to use the RPC client and server programs
// Author : Neha Tilak

// STEPS :

1. Start server using the command:
    ./server.run

2. Start client using the command
   ./client.run

3. You can change the client commands by changing the file 'commandlist.txt'

4. You can change the server config by changing the file 'ServerConfig.json'

5. You can change the client config by changing the file 'ClientConfig.json'

For details regarding implementation and design, see the project report :
            NehaTilak_Project1.doc
